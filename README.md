## Local Build

1. Clone the GitHub repository

   ```bash
   git clone git@github.com:sunillshastry/sunilshastry.com.git
   ```

2. Manually `cd` into directory/repository, simply open the `index.html` entry source file from the directory.

   ```bash
   cd sunilshastry.com
   open index.html
   ```
